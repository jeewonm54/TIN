### 연습문제 5. 동명 이인이 있는 이름은 무엇일까요? 

![week3_sql5.png]()

- 집계 후 조건 => HAVING, FROM
- WHERE : 원본 데이터 FROM 절에 있는 데이터에 조건을 설정하고 싶은 경우
- HAVING : GROUP BY 와 함께 집계 결과에 조건을 설정하고 싶은 경우 
- 서브쿼리 : 쿼리문을 한번 감싸서 다른 쿼리문에서 사용할 수 있음
- HAVING 을 쓰면 쿼리 줄 수가 줄어듦
